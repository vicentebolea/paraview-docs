3 Introduction
4 ============
5 [ParaView][] is an open-source, multi-platform data analysis and
6 visualization application based on
7 [Visualization Toolkit (VTK)][VTK].
8 
9 The first public release was announced in October 2002. Since then, the project
10 has grown through collaborative efforts between [Kitware Inc.][Kitware],
11 [Sandia National Laboratories][Sandia],
12 [Los Alamos National Laboratory][LANL],
13 [Army Research Laboratory][ARL], and various other
14 government and commercial institutions, and academic partners.

47 Building
48 ========
49 
50 There are two ways to build ParaView:
51 
52 * The easiest method for begginners to build ParaView from source is
53 by using our [Getting Started compilation guide][build] which includes
54 commands to install the needed dependencies for most operating systems.
55 
56 * Another way to build ParaView, quite useful when trying to enable more specific
57 options which requires to build dependencies yourself (ie osmesa, raytracing),
58 would be the [ParaView Superbuild][sbrepo]. The superbuild downloads and builds all
59 of ParaView's dependencies as well as ParaView itself.
60 
61 [sbrepo]: https://gitlab.kitware.com/paraview/paraview-superbuild
62 [build]: Documentation/dev/build.md

64 Reporting Bugs
65 ==============
66 
67 If you have found a bug:
68 
69 1. If you have a source-code fix, please read the [CONTRIBUTING.md][] document.
70 
71 2. Otherwise, please join the [ParaView Discourse][] forum and ask about
72  the expected and observed behaviors to determine if it is really a bug.
73 
74 3. Finally, if the issue is not resolved by the above steps, open
75  an entry in the [ParaView Issue Tracker][].
76 
77 [ParaView Issue Tracker]: https://gitlab.kitware.com/paraview/paraview/-/issues
