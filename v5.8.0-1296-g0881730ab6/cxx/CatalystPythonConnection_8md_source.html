1 ## ParaView catalyst live python API
2 
3 ParaView now provides a python API to create live visualization with Catalyst.
4 The new `live` module includes the following method:
5 * `ConnectToCatalyst` to initiate communication with a running Catalyst server
6 * `ExtractCatalystData` to extract data from catalyst to client.
7 * `PauseCatalyst` to pause the simulation
8 * `ProcessServerNotifications` to look for catalyst notifications.
9 
10 You can find an example of use in ParaView sources `Examples/PythonClient/catalystLiveVisualizer.py`
