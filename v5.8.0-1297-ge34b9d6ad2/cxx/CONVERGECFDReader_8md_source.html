1 ## New CONVERGE CFD post reader
2 
3 A reader for CONVERGE CFD post files has been added. This reader reads files containing
4 meshes, surfaces, and parcels. Each stream in a file is read as a top-level
5 block and meshes, surfaces, and parcels are datasets under each stream block.
6 
7 Cell data arrays associated with mesh cells can be individually
8 selected for reading using the CellArrayStatus API.
9 
10 Point data arrays associated with parcels can be individually selected
11 for reading using the ParcelArrayStatus API.
12 
13 Time series are supported. The reader assumes a time series is defined
14 in a sequence of files that follow the naming convention
15 
16 `<prefix><zero-padded index>[_][<time>].h5`
17 
18 where the prefix is determined from the FileName property passed to
19 the reader. The underscore and time elements are optional. The time
20 value associated with each file is read from metadata in the file.
