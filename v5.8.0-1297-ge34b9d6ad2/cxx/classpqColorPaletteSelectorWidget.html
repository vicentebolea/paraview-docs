Detailed Description

widget to choose a color palette to load/select.

pqColorPaletteSelectorWidget is a pqPropertyWidget intended to be used in two roles:

  1. To load a specific color palette.
  2. To select a specific color palette.

Mode (1) is used when the widget is used for a vtkSMProperty e.g. **'Load Palette'** property on the ColorPalette proxy. In that case, the user's action is expected to update the proxy with the chosen palette.

Mode (2) is used when the widget is used for a vtkSMStringVectorProperty e.g. OverrideColorPalette property on ImageOptions proxy. In that case, the selected palette name is simply set on the vtkSMStringVectorProperty.
