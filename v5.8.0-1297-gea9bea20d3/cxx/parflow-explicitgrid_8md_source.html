3 The ParFlow plugin now uses VTK's explicit structured grid[1] data format
4 to represent the subsurface simulation domain when terrain deflection is
5 turned on (i.e., when deforming the grid by elevation). This increases
6 memory usage but provides visualizations more useful in geology where
7 discontinuities are often present.
