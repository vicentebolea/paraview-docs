Detailed Description

controller for extract generation

vtkSMExtractsController is a controllers that provides API to handle various aspects of extract generators and extract generation mechanisms supported by ParaView.

It provides API to query, create extract generators of known types. It also provides API to generate extracts using the defined extract generators.
