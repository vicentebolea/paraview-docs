1 # Improvements to spreadsheet view for multi-block datasets
2 
3 Spreadsheet view now shows names of blocks when showing
4 composite datasets in a separate column. This makes it easier to know
5 which row corresponds to which block.
6 
7 Paritial arrays i.e. arrays not present on all blocks are now supported. If an
8 array is absent on some of the blocks, the correspoding cell will be shown as
9 empty. Previously, such paritial arrays were not shown at all in the view.
