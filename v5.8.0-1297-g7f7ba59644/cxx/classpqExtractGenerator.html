Detailed Description

pqProxy subclass for extract generators

pqExtractGenerator is an Qt-adaptor for an extract generator proxy. Extract generators are used in ParaView to generate data or image extracts. This class is used to represent such a proxy. It simply provides convenience API to make it easier to monitor an extract generator proxy in the Qt code.
